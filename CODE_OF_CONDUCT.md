# 貢獻者行為準則 (Contributor Covenant Code of Conduct)

## 我們的承諾

身為本社群的成員、參與者與貢獻者，我們承諾為每個人提供一個免於騷擾的參與體驗，不分年齡、體型、可見或不可見的身心障礙、種族、性特徵、性別認同與表達、經驗水平、教育程度、社會經濟地位、國籍、外貌、人種、階級、宗教或性取向。

我們承諾以有助於建立開放、歡迎、多元、包容且健康的社群方式進行互動。

## 我們的標準

有助於為社群創造積極環境的行為範例包括：

* 對他人表現出同理心與善意。
* 尊重不同的意見、觀點與經驗。
* 提供並優雅地接受建設性的反饋。
* 對受到我們錯誤影響的人承擔責任並道歉，並從經驗中學習。
* 關注什麼對整體社群最好，而不僅僅是個人。

不可接受的行為範例包括：

* 使用性暗示語言或影像，以及不受歡迎的性關注或挑逗。
* 挑釁、侮辱或貶低性的言論，以及人身或政治攻擊。
* 公開或私下的騷擾。
* 未經明確許可發布他人的私密資訊（如地址或電子郵件）。
* 其他在專業環境中可被視為不當的行為。

## 執行的責任

社群負責人有責任澄清並執行可接受行為的標準，並將針對任何其認為不當、威脅、攻擊或有害的行為採取適當且公平的糾正措施。

社群負責人有權利且有責任刪除、編輯或拒絕不符合本行為準則的評論、提交 (Commits)、程式碼、Wiki 編輯、Issues 以及其他貢獻，並在適當時溝通處理原因。

## 適用範圍

本行為準則適用於所有社群空間，也適用於個人代表社群在公共空間進行溝通的情況。代表社群的例子包括使用官方電子郵件地址、透過官方社群媒體帳號發文，或在線上、線下活動中作為指定的代表。

## 執行流程

如發現具有虐待性、騷擾性或其他不可接受的行為，請向社群負責人舉報：[請填入您的聯絡郵件]。所有投訴都將得到及時且公正的審查與調查。

所有社群負責人均有義務尊重任何事件舉報者的隱私與安全。
